# Simple Interest Calculator Project

## Project Description
This project is a Simple Interest Calculator built using a Bash script. It allows users to calculate simple interest based on user input values.

## How it Works
The calculator takes three inputs from the user:
- Principal amount (P)
- Rate of interest (R)
- Time period in years (T)

It then calculates simple interest using the formula:

Simple Interest = (P × R × T) / 100

## Features
- Takes user input from the terminal
- Performs automatic calculation
- Displays formatted output
- Easy to use Bash script
